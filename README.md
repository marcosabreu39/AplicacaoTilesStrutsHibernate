# Aplicação com Tiles, Struts e Hibernate

## Tecnologias

* Arquitetura MVC;
* Design Pattern Facade;
* Design Pattern Repository;
* Design Pattern DAO;
* Apache Tiles 3;
* Apache Struts 2;
* JPA2(Hibernate 5);
* CDI 1.2;
* Bootstrap 4;
* Maven;
* Log4j;
* Banco de dados PostgreSQL 9;
* Servidor Apache Tomcat 9;
* IDE Eclipse.
